# Nuxt sample

The base template with Nuxt.

## Get started 🚀

### 🐋 Docker

If you use docker, you can run `docker compose up -d` in the working directory.

```sh
docker compose up -d
```

Then you can access to `http://localhost:3000`

### 😺 Yarn

If you don't use docker, you need to run below command.

```sh
yarn
yarn dev
```
